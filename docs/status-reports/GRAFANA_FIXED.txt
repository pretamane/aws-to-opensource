WHAT WAS FIXED


Issue: Grafana and Prometheus not accessible via reverse proxy

Root Cause:
  - Grafana needed GF_SERVER_SERVE_FROM_SUB_PATH configuration
  - Prometheus needed prefix stripping in Caddy
  - Meilisearch needed prefix stripping in Caddy
  - MinIO needed prefix stripping in Caddy

Solutions Applied:
  1. Grafana: Added subpath environment variables
     - GF_SERVER_ROOT_URL=http://localhost/grafana
     - GF_SERVER_SERVE_FROM_SUB_PATH=true

  2. Prometheus: Used handle_path in Caddy to strip /prometheus prefix
  
  3. Meilisearch: Used handle_path in Caddy to strip /meilisearch prefix
  
  4. MinIO: Used handle_path in Caddy to strip /minio prefix

Result:  All monitoring services now accessible!
